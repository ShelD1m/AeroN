# AeroN

**AeroN** — это сервис для мониторинга воздушного трафика в реальном времени.  
Приложение интегрируется с **OpenSky Network API** и предоставляет REST-интерфейс для получения данных о текущих полётах, включая идентификатор, координаты, скорость и другую информацию о самолётах.

---

## Описание проекта

AeroN — это бэкенд-сервис, реализованный на **Spring Boot**, который собирает данные из внешнего источника (OpenSky API) и предоставляет удобный REST-фид для последующей визуализации или аналитики.  
Сервис может использоваться в качестве базы для построения панелей мониторинга воздушного движения, интеграции с системами навигации или анализа авиаперелётов.

---

## Основные возможности

- Получение данных о всех активных самолётах в реальном времени  
- REST-эндпоинт `/Feed/AircraftList.json`  
- Поддержка CORS для интеграции с внешними фронтенд-приложениями  
- Логическая структура: **Controller → Service → Model → Data Layer**  
- Конфигурируемое подключение к OpenSky API  
- Возможность масштабирования и интеграции с другими источниками данных  

---

## Технологии и стек

- **Java 17+**
- **Spring Boot** (Web, Configuration, Dependency Injection)
- **Maven**
- **REST API**
- **OpenSky Network API**
- **JSON Serialization**
- **CORS support**

## Запуск проекта

### Через Maven

```bash
mvn spring-boot:run
```

### Или через собранный jar

```bash
mvn clean package
java -jar target/AeroN-0.0.1-SNAPSHOT.jar
```

После запуска API будет доступно по адресу:  
`http://localhost:8080/Feed/AircraftList.json`
